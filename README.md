# CdSO-Shell
progetto scelto:

  - shell (con FS)
   - File mmappato E persistente
   - Comandi usuali di shell (e di "libreria)
   - format <fs_filename> <size>
   - mkdir
   - cd
   - touch (crea un file vuoto)
   - cat (stampa il contenuto di un file su schermo)
   - ls
   - append <file> <testo>
   - rm <dir/file>
   - close
